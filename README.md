## ⚒️ TroubleShooting
Windows Forms 을 개발을 하려면 윈도우 OS에서 작업을 해야해서 MAC 환경에서 작업중이던 쇼핑몰 프로젝트를 개인 서버로 돌렸습니다.
<br>
MAC 환경에서 돌아가고 있는 MYSQL을 접속하려고 했지만 실패하였다.
<br>
